# Customer Churn Prediction

## Project Overview

This project predicts whether a customer is likely to leave a telecom service using Machine Learning.

Customer churn prediction helps businesses identify at-risk customers and improve customer retention strategies.

---

## Dataset

IBM Telco Customer Churn Dataset

- Total Records: 7043
- Features: 21
- Target Variable: Churn

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook
- Random Forest Classifier

---

## Project Workflow

1. Data Cleaning
2. Missing Value Handling
3. Feature Encoding
4. Train-Test Split
5. Model Training
6. Prediction
7. Model Evaluation

---

## Machine Learning Model

Random Forest Classifier

Reason:
- Handles large datasets efficiently
- Reduces overfitting
- Provides good classification performance

---

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

Since customer churn prediction is a business-critical problem where missing a churn customer can result in revenue loss, greater emphasis was placed on Recall and F1-Score.

---

## Key Features Affecting Churn

- Contract Type
- Tenure
- Monthly Charges
- Total Charges
- Internet Service
- Tech Support

---

## Project Outcome

The model successfully predicts whether a customer is likely to churn based on historical customer data.
